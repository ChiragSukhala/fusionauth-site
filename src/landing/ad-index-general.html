FusionAuth

Registration, login, SSO, MFA, social login, user management, reporting, analytics and much more. FREE for unlimited users.

FusionAuth installs in minutes, and provides enterprise-level scalability plus 24/7 support plans, and managed cloud hosting options.

Run FusionAuth on any platform

Install in seconds with a single command.

  • Linux
  • macOS
  • Windows
  • Docker

Copy and paste the code below into Terminal and we'll do the rest.

Installs in your current working directory using ZIP packages

sh -c "curl -fsSL https://raw.githubusercontent.com/FusionAuth/fusionauth-install/master/install.sh | sh"

Or install using Homebrew using the following commands.

Install the FusionAuth tap, this will only need to be performed once

brew tap fusionauth/homebrew-fusionauth

Install FusionAuth

brew install fusionauth-app fusionauth-search

Start FusionAuth

brew services start fusionauth-search
brew services start fusionauth-app

Copy and paste the code below into Terminal and we'll do the rest.

Installs in your current working directory using ZIP packages

sh -c "curl -fsSL https://raw.githubusercontent.com/FusionAuth/fusionauth-install/master/install.sh | sh -s - -z"

Installs for all users on the system using DEB or RPM packages

sh -c "curl -fsSL https://raw.githubusercontent.com/FusionAuth/fusionauth-install/master/install.sh | sh"

Copy and paste the code below into Powershell and we'll do the rest.

Installs in your current working directory using ZIP packages

iex (new-object net.webclient).downloadstring('https://raw.githubusercontent.com/FusionAuth/fusionauth-install/master/install.ps1')

Use the following Docker commands to get up and running quickly. Please feel free to open an issue or submit a pull request in GitHub if you find an issue. 

Use Docker Compose to build an entire FusionAuth instance

curl -o docker-compose.yml https://raw.githubusercontent.com/FusionAuth/fusionauth-containers/master/docker/fusionauth/docker-compose.yml && curl -o .env https://raw.githubusercontent.com/FusionAuth/fusionauth-containers/master/docker/fusionauth/.env && docker-compose up

Installs only FusionAuth App in the container

docker pull fusionauth/fusionauth-app

All images can be found on Docker Hub. Review our Docker Installation guide for additional information.
