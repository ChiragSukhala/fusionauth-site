== Go Client Library

The Go client library allows you to integrate FusionAuth with your Go application.

Source Code:

* https://github.com/FusionAuth/fusionauth-go-client

Installation

```bash
go get github.com/FusionAuth/fusionauth-go-client/pkg/fusionauth
```
